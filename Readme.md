# NRV-BLOG

## 1. Download The Project or Git Clone it.
```
git clone https://github.com/niruparekh09/NRV-Blog
```

## 2. To Run The Project
### To Start Frontend 
```
cd BlogUI
npm i
npm run dev
```
### To Start Backend
```
cd Nrv-Blog-API
mvn clean install
mvn spring-boot:run
```
## 3. You can also import postman file
### `Nrv-Blog-API.postman_collection.json`
